"# Facial_Recognition" 
This program draws dataset from the website www.Kaggle.com (available for free). The purpose of the program is to display, the name of the famous personality upon test trial on a random photo (by pixel-by-pixel comparision with that of the training set). The algorithm used here is KNN, alongside the diligent use of haarcascade frontal face classifier.

The program titled "video_stream_img_detect.ipynb" is a program meant to detect faces from live videostream (default- laptop camera).

This project is totally for educational purpose and the author/ any of its primary contributor(s) have no involvement in any of its commercial use.
